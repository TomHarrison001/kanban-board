# React Native + Expo

This is an application built using:

- React Native
- Expo
- TypeScript
- Convex

## Features

- Create todos
- Edit/Update todos
- Delete/Complete todos
- Dark mode
- Progress bar

## Get started

1. Clone repository
2. Install dependencies `npm install`
3. Start the app `npx expo start`
4. Open http://localhost:8081/ to view the app in web mode
