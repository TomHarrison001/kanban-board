# React Native + Expo

This is an application built using:

- React Native
- Expo
- TypeScript

## Features

- (tabs) directory
- styled index screen
- styled settings screen
- multiple tabs screens

## Get started

1. Clone repository
2. Install dependencies `npm install`
3. Start the app `npx expo start`
4. Open http://localhost:8081/ to view the app in web mode
